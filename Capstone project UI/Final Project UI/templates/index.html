Pipeline

  1. Sample evenly spaced frames from the uploaded video.
  2. Phase 1 computes residual and FFT features for AI vs camera classification.
  3. If Phase 1 predicts camera-captured, Phase 2 extracts face crops and runs the ensemble.
  4. The final output becomes one of: AI Generated, Real, or Deepfake.
